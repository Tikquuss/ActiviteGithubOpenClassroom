Il s'agit d'un projet java qui affiche les informations d'un utilisateur au clavier
Un utilisateur est caractérisé par son nom(obligatoire) et son prenom
Au lancement du programme (javac TestUtilisateur.java; java TestUtilisateur) le nom et le prenom sont demandés, en suite ils sont affichés dans la console java.
Cette fois si le nom est demandé tant que le nom entré est vide, car le nom est obligatoire
